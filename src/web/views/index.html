Bot Features

AI Assistant

Available Commands

!help - Get Help

Shows all available commands and how to use them

!ping - Check Status

Checks if the bot is running and shows uptime

!ai - AI Assistant

Ask the AI anything - it will respond with helpful information

!image - Image Generation

Generates images based on your text description (admin only)

!link allow - Link Management

Add domains to the whitelist (admin only)

!ban - User Moderation

Ban users from the group (admin only)
